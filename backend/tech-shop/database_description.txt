👤 1. USERS
👉 Chức năng: Lưu thông tin tài khoản người dùng
| Thuộc tính   | Kiểu         | Mô tả                    |
| ------------ | ------------ | ------------------------ |
| user_id (PK) | CHAR(36)     | ID người dùng (UUID)     |
| role_id (FK) | INT          | Vai trò (admin/user)     |
| email        | VARCHAR(255) | Email đăng nhập (unique) |
| password     | VARCHAR(255) | Mật khẩu đã mã hóa       |
| full_name    | VARCHAR(120) | Họ tên                   |
| phone        | VARCHAR(20)  | Số điện thoại            |
| created_at   | DATETIME     | Ngày tạo tài khoản       |
